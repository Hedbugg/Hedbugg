## 📌 Featured Projects

### 🔹 Raw Materials Inventory Management System (IMS)
Full internal system for a concrete/construction materials company — dashboard with charts and stat tiles, Purchase Request module with print-ready A5 forms (Khmer text + branding), role-based access control, and a reusable component architecture.
- **Stack:** React, TypeScript, Tailwind, MUI, Zustand, TanStack Query
- **Backend:** Go / Gin / PostgreSQL / MongoDB
- **Highlights:** ERP-style data grids, custom `useTranslation` i18n system, JWT auth with persisted state

### 🔹 Room Renter System
Full-stack project focused on real-world data handling and CRUD-heavy workflows.
